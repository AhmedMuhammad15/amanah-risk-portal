# AI-Driven Shariah Compliance & Credit Risk Portal

An enterprise-grade FinTech application designed for automated Islamic banking (Murabaha) underwriting. This portal screens applicants for Shariah compliance and uses Machine Learning to predict credit default risk based on operational financial metrics.

## 🚀 Key Features

*   **Dual-Gate Evaluation Engine:** Automatically intercepts applications if the business sector or target asset violates Shariah compliance parameters.
*   **Machine Learning Underwriting:** Integrates an Ensemble Random Forest Classifier to process financial inputs (Income, Debts, Tenure) and yield a real-time default probability score.
*   **Interactive Professional UI:** Built with a clean, executive-level corporate dashboard layout inspired by premium financial web applications.
*   **Advanced Accessibility Controls:** 
    *   **Dynamic Localization:** Toggle instantly between English and Urdu layouts with automated text direction (RTL/LTR) adjustment.
    *   **Font Scaler:** Quick 'A+' and 'A-' responsive text scaling for users across all age groups.
    *   **Theme Switcher:** Seamless transition between high-contrast Dark Mode and clean Light Mode.

## 🛠️ Tech Stack

*   **Frontend & Layout:** Streamlit, Custom HTML5 & CSS3 Injectors
*   **Data Processing & Modeling:** Python, Pandas, NumPy, Scikit-Learn (Random Forest)
*   **Model Serialization:** Pickle Pipeline
